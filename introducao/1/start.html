Antes de começarmos a falar em lógica de programação, é importante entender alguns conceitos básicos!

Primeiramente, precisamos entender que um computador é composto basicamente por duas partes essenciais que trabalham em conjunto.

Temos a parte física, como por exemplo o smartphone que está sendo usado e seus componentes internos. Essa "parte" chamamos de Hardware

E temos a parte lógica, que chamamos de

SOFTWARE

Cada "programa" do seu computador é um software!

Este aplicativo é um software!

O Sistema Operacional do seu computador ou celular (Android, iOS, Windows, etc...) também é um software!

Para facilitar, podemos comparar o conceito de Hardware e Software com a mente humana.
Nosso cérebro seria o Hardware!
Nossos pensamentos seriam o Software!

Para que o software exista, alguém precisa antes escrever um algoritmo, seguindo uma determinada linguagem e alguns padrões de lógica, que poderá ser interpretado pelos nossos computadores.

Nosso objetivo então é focar nessa lógica, para que no futuro você seja capaz de construir aplicações para as mais variadas plataformas!
